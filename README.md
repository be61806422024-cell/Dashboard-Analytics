# SACCO Financial Dashboard

Interactive dashboard for KEMU SACCO's Statement of Comprehensive Income (2021–2025).

## Features
- Revenue vs Expenditure trend
- Net surplus and dividend analysis
- Key financial ratios (cost of funds, net margin, dividend payout)
- Revenue composition by source
- Expense breakdown
- Download filtered data as CSV

## Live Demo
[Deploy on Streamlit Cloud](https://streamlit.io/cloud)

## Local Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## Data Source
SACCO audited annual reports (2021–2025).
